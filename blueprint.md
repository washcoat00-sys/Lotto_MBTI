
# Lotto Number Generator & MBTI Assessment

## Overview

This is a web application that includes a lottery number generator and an MBTI personality assessment tool.

## Features

*   **Lotto Generator:**
    *   Generates 6 unique random numbers between 1 and 45.
    *   Displays the numbers in a visually appealing way.
    *   A button to generate a new set of numbers.
*   **MBTI Assessment:**
    *   Presents a series of questions to determine the user's MBTI type.
    *   Calculates and displays the MBTI result based on the user's answers.
